# component-cy
基于Angular.js与Vue.js公司自主开发的一套框架，在新框架的基础上，开发出来的一套前端常用组件


# 组件化开发的优点
> 1.块状化结构，减少css 的书写，并且方便扩展

> 2.标记鲜明，容易维护



# 主要组件
## 基础组件
>1. 按钮组件

>2. SASS编译 压缩 合并

>3. JS合并 压缩 混淆

>4. img压缩

>5. html压缩

>6. 代码实时监测


## 使用步骤


```bash
$ git clone https://github.com/chenya1123236324/gulp-cy.git
$ npm install
$ gulp
```
